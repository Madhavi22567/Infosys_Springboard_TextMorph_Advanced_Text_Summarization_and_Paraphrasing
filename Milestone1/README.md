## Description
In this milestone, we developed a secure user authentication system using Streamlit, JWT, and

- Secure Login using JWT
- Dashboard after login
- Forgot Password functionality
- Ngrok integration
## How to Run
1. Install dependencies
2. Run Streamlit app
3. Use ngrok to expose the app
